# JetKit

SAAS kit: TRPC API + Auth + Vue SPA

- TypeScript
- Postgres ([Drizzle](https://orm.drizzle.team/))
- [TRPC](https://trpc.io/)
- [Hono](https://hono.dev/)
- [BetterAuth](https://www.better-auth.com/)
- [Vite](https://vite.dev/)
- [Vue 3](https://vuejs.org/)
- [Zod](https://zod.dev/)
- [Tailwind CSS](https://tailwindcss.com/) (and [DaisyUi](https://daisyui.com/))
